    @mixin colors($colors: $theme-colors) { ... }

    Description

    Quickly and easily create color presentational classes. Includes background-color, color, and fill clases.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $colors

    The theme color map.

    		Map$theme-colors

    Example

    Basic usage.

    @include colors($theme-colors);

    Output

    .background-blue {
  background-color: #00f;
}
.color-blue {
  color: #00f;
}
.fill-blue {
  fill: #00f;
}

    Throws

    • Sorry, $colors is not a Sass map!

    font-legibility

    @mixin font-legibility() { ... }

    Description

    Improve the font rendering to better match client PSDs. Side effects include: slower rendering of pages. Use only when necessary!

    Parameters

    None.

    Example

    Basic Usage Sass

    .foo {
    @include font-legibility;
}

      text-rendering: optimizeLegibility;
      -moz-font-feature-settings: 'liga', 'kern';
      -moz-osx-font-smoothing: grayscale;
      -webkit-font-smoothing: antialiased;
}
